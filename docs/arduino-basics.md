# Arduino の基礎

## 言語と開発環境

- Arduino のコードは **C++** で動きます。
- 開発は基本的に **Arduino IDE** をベースに行います（コードの編集・コンパイル・書き込み）。
- 書いたコードの保存・バージョン管理は **Git** で行います。

```text
Arduino IDE で編集
↓ コンパイル & 書き込み（USB）
Arduino UNO で実行
↓
コードの保存・管理は Git
```

## プログラムの基本構造

Arduino のスケッチ（プログラム）は、最低限この2つの関数で構成されます。

```cpp
void setup() {
  // 起動時に1回だけ実行される
}

void loop() {
  // 電源が入っている間、ずっと繰り返される
}
```

### setup()

起動時に **1回だけ** 実行される関数。主に初期設定を書きます。

```cpp
void setup() {
  pinMode(13, OUTPUT);   // 13番ピンを出力にする
  Serial.begin(9600);    // シリアル通信を開始する
}
```

よく書く処理:

- ピンの入出力設定（`pinMode`）
- シリアル通信の開始（`Serial.begin`）
- サーボやセンサーの初期化

> PCアプリで言う「初期化処理」にあたります。

### loop()

電源が入っている間、**ずっと繰り返される** 関数。実際の動作を書きます。

```cpp
void loop() {
  digitalWrite(13, HIGH);  // LED ON
  delay(1000);             // 1秒待つ
  digitalWrite(13, LOW);   // LED OFF
  delay(1000);             // 1秒待つ
}                          // 最初に戻って繰り返す
```

Arduino には普通の OS がありません。
そのため、プログラムは基本的に `loop()` の中をずっと回り続けます。

```text
電源ON
↓
setup() を1回実行
↓
loop() を無限に繰り返す
```
