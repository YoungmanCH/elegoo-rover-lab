# 自作ルンバの開発（おもちゃにウェットシートを引きずらせてみた）

> ロボットにウェットシートを引きずらせて床を拭く、自走式のお掃除ロボ。決めた走り方で部屋を回り、壁は超音波センサで検知して曲がる。これをブラウザから操作する。

## 1. これは何か

要素は3つだけ。

- **機構**：車体にウェットシートを付けて引きずる。
- **自走**：決めた走り方（経路）で床を回る。壁は超音波で検知して曲がる。
- **Web**：ブラウザから走行の ON/OFF・操作をする。

```
[ブラウザ] ──操作──▶ [ロボット：自走＋壁検知] ──引きずる──▶ [ウェットシート]
```

位置を測る手段（車輪エンコーダ）が無いので、「経路」は地図上の線ではなく **「壁を検知したら曲がる」という走り方の決まり** として持つ。これで「部屋全体をざっくり」をシンプルに実現する。

## 2. 最初にやること：経路（走り方）のテスト設計

Web より先。ここが土台。

- **範囲**：どの広さを対象にするか（例：フローリングの一室）。
- **走り方**：壁まで直進 → 旋回 → また直進、の繰り返しを基本にする（往復／ジグザグ風）。
- **テスト**：実機でその走り方を流し、①どれだけ床をカバーするか ②壁検知で破綻しないか（ぶつかる／止まる／同じ所をぐるぐる）を見る。

走り方が決まれば、自走ロジックも Web 操作もその上に乗るだけ。

## 3. 進め方（ざっくり）

1. **経路テスト** — 走り方を決めて実機で検証（自走の核）
2. **Web 操作** — ブラウザから走行 ON/OFF・手動操作
3. **（余れば）状態表示** — 距離などを Web にライブ表示
